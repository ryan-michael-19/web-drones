Web Drones!

Like playing games?

In Web Drones you explore the world looking for scrap metal. Your drones use this scrap to make themselves more companion drones.

Like writing code?

Web drones are controled via a REST api, allowing players to learn how to make HTTP requests and write code while playing the game.

-

Web Drones is in pre-alpha, and breaking changes WILL be deployed regularly. Check this page to see if any affect you!

The current rate limit per account is set to 2 requests per second.

All users refresh every Sunday at 8 AM!

Already created an account and written a script or two? View the radar to see what your drones are up to! You can also build some frontend skills by using the Web Drones API to create your own radar, or any other visualization you can think of!
